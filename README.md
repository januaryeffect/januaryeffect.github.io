# January Effect 회사 홈페이지

(주)제뉴어리이펙트 공식 홈페이지입니다.

## 기술 스택

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Hosting**: GitHub Pages

## 개발

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```
